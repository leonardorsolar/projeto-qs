# Análise

## Requisitos Funcionais:

# 1-Login de Usuário:

O sistema deve permitir que usuários registrados façam login utilizando e-mail e senha.
Deve haver um mecanismo de validação para verificar a autenticidade dos dados inseridos.

# 2-Registro de Usuário:

O sistema deve permitir que novos usuários se cadastrem inserindo nome, e-mail, senha, e confirmação de senha.
As senhas devem ter requisitos mínimos, como ao menos 8 caracteres, incluindo números e letras.

# 3-Recuperação de Senha:

O sistema deve permitir que usuários solicitem a recuperação de senha, enviando um e-mail de redefinição.

# 4-Dashboard de Conta:

Após o login, os usuários devem ter acesso a um painel que exibe o saldo da conta e as últimas transações.
O usuário deve poder consultar detalhes de sua conta e editar informações pessoais (e-mail, senha, etc.).

# 5-Transações Bancárias:

O sistema deve permitir que o usuário visualize o histórico de transações bancárias, incluindo depósitos e saques.
O usuário deve poder transferir valores para outra conta, inserindo o número da conta destino e o valor.

# 6-Logout:

O sistema deve permitir que o usuário saia de sua conta a qualquer momento com a opção de logout.
